# Cursor UI Clone (HTML & CSS)

## Overview
This project is a simple static recreation inspired by the Cursor website.  
It focuses on rebuilding selected sections using only HTML and CSS to practice layout, structure, and styling.

## Sections Recreated
- Navigation bar
- Hero section
- Features section
- Testimonials section
- Footer

All sections are static and do not include JavaScript-based interactivity.
